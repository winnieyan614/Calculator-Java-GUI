Calculator-Java-GUI
===================

A calculator I created using Java

Calculator functions:
- Non-scientific calculator
- Scientific calculator including:
  - Sin
  - Sin-1
  - Cos
  - Cos-1
  - Tan
  - Tan-1
  - x^2
  - x^3
  - Square Root
  - Cube Root
  - Ln
  - LOG
  - 1/x
  - +/-
  - Memory
